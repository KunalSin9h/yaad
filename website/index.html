features

Everything you need,
nothing you don't.

Built for developers who live in the terminal and treat their privacy seriously.

recall

Natural language search

Ask in plain English. Local embeddings + LLM reasoning find exactly what you saved, even weeks later.

privacy

Fully local & private

All AI runs via Ollama on your machine. Zero cloud, zero accounts, zero telemetry. Your data never leaves.

reminders

Smart time parsing

Say "in 30 minutes", "tomorrow 9am", or "Friday 3pm". yaad turns it into a real deadline.

context

Auto-enriched metadata

Every memory captures cwd, hostname, and timestamp. AI auto-detects type and extracts tags for you.

resilience

Offline-safe

Memories save even when Ollama is unavailable. AI enrichment degrades gracefully — your data is never lost.

architecture

Fully extensible

Ports & Adapters design. Swap the storage backend, AI provider, or notifier by implementing one interface.

installation

Up and running in minutes.

Requires Go 1.21+ and Ollama running locally.

1

Install Ollama and pull models

Get Ollama from ollama.com, then pull the required models.

# embeddings model
ollama pull nomic-embed-text
# chat / reasoning model (swap for any compatible one)
ollama pull llama3.2:3b
2

Install yaad

go install github.com/kunalsin9h/yaad/cmd/yaad@latest
3

Initialise config

yaad config init
4

Save your first memory

yaad add "hello world" --for "my first yaad memory"
usage

A command for every moment.

Five memory types, auto-detected by AI: command · note · url · fact · reminder

yaad add "<content>" [flags]

Save a memory. Use --for for context, --remind for a deadline, --tag for labels.

yaad ask "<question>"

Query everything with natural language. The LLM reasons over semantic search results.

yaad list [flags]

Browse recent memories. Filter by --type, --tag, or --remind.

yaad get <id>

Show full details of a memory. Accepts a 10-char ULID prefix.

yaad delete <id>

Remove a memory by ID. Prompts for confirmation; skip with -y.

yaad config init / set / get

Manage ~/.yaadrc. Override Ollama URL, models, notifier, and poll interval.

# save a command you'll want to resume
yaad add "claude --resume 17a43487" --for "yaad build session"

# a fact with tags
yaad add "staging postgres is on port 5433" --tag postgres --tag staging

# a time-sensitive reminder
yaad add "submit PR for review" --remind "tomorrow 9am"

# recall anything naturally
yaad ask "what was the staging postgres port?"
yaad ask "do I have anything due tonight?"
reminders

Never miss a deadline.

Two ways to surface reminders — pick what fits your workflow.

recommended

Inline — PROMPT_COMMAND

Reminders surface in your terminal on every prompt. No background process needed.

# ~/.bashrc
export PROMPT_COMMAND="yaad check; $PROMPT_COMMAND"

# ~/.zshrc
precmd() { yaad check }
systemd

Background daemon

Install as a systemd user service. Desktop notifications via notify-send.

yaad daemon install
systemctl --user enable --now yaad

# check status
systemctl --user status yaad
